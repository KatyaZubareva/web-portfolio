# Что было сделано

## В рамках задания я:

Изучила материалы по развертыванию статического сайта через GitHub Pages.

Установила CMS Hugo локально (через Homebrew / бинарный архив).

Создала новый Hugo-проект, добавила 2 страницы контента (/content/about.md и /content/projects.md).

Выполнила локальную сборку проекта командой hugo — сгенерированный сайт появился в каталоге /public.

Результат сборки загрузила вручную на GitHub Pages в отдельный репозиторий (ветка gh-pages).

Исходники Hugo-проекта разместила в отдельном репозитории.

Настроила GitHub Actions, чтобы автоматически собирать Hugo-сайт при каждом пуше и публиковать в ветку gh-pages.
