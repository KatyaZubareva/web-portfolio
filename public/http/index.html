HTTP Protocol. Client–Server Interaction

Практическая работа по изучению HTTP-запросов и инструментов взаимодействия с API
Катя Зубарева

На этой странице представлен отчёт по лабораторной работе, посвящённой протоколу HTTP и клиент–серверному взаимодействию.
Работа включает теоретическое изучение HTTP и практическую отправку GET и POST запросов с использованием CLI и GUI-инструментов.

Цель работы#

Изучить структуру HTTP-запросов и ответов, освоить отправку запросов различными инструментами и получить практический опыт работы с публичными API.

Теоретическая часть#

HTTP (HyperText Transfer Protocol) — прикладной протокол передачи данных, используемый для взаимодействия клиента и сервера в веб-приложениях.

Основные элементы HTTP-запроса#

  • метод (GET, POST и др.)
  • URL
  • версия протокола
  • заголовки (Headers)
  • тело запроса (Body, опционально)

Основные элементы HTTP-ответа#

  • код состояния (200, 404, 500 и др.)
  • заголовки
  • тело ответа

Практическая часть#

Отправка GET-запроса через Telnet#

Описание: ручная отправка HTTP-запроса на веб-сервер через Telnet.
Задачи: изучить низкоуровневую структуру HTTP-запроса и ответа.
Инструменты: Telnet.

Результат: получен HTTP-ответ со статусом 200 OK и HTML-контентом страницы.

(Скриншот подключения и ответа сервера)

Отправка POST-запроса через netcat#

Описание: отправка POST-запроса с JSON-телом на локальный сервер.
Задачи: передача данных в теле HTTP-запроса, работа с заголовками.
Инструменты: netcat.

Пример данных:

  • book: «Чапаев и Пустота»
  • author: Виктор Пелевин
  • price: 100

Результат: сервер принял данные и вернул подтверждающий ответ.

(Скриншот запроса и ответа)

Отправка запросов с помощью cURL#

Описание: выполнение POST-запроса к API через CLI.
Задачи: освоить синтаксис cURL, работу с заголовками и JSON.
Инструменты: cURL.

Результат: получен JSON-ответ сервера, подтверждающий успешную обработку запроса.

(Скриншот терминала с выполненным запросом)

Использование GUI-инструмента#

Получение курса валют через API Банка России#

Описание: выполнение GET-запроса к XML API Банка России для получения курса валюты за выбранный период.
Задачи: работа с публичным API, параметрами запроса и XML-ответом.
Инструменты: Postman / Insomnia / HTTPie Desktop.

Пример параметров запроса:

  • период дат
  • код валюты (USD)

Результат: получен XML-документ с курсами валют за указанный период.

(Скриншот запроса и ответа в GUI-инструменте)

Используемые инструменты#

  • Telnet
  • netcat
  • cURL
  • Postman / Insomnia
  • API Банка России

Выводы#

В ходе работы были изучены основы протокола HTTP и реализованы GET и POST запросы с использованием различных инструментов. Работа позволила на практике увидеть различия между низкоуровневым и высокоуровневым взаимодействием с HTTP и закрепить понимание клиент–серверной архитектуры.

В разработке#

В дальнейшем планируется изучение:

  • аутентификации и авторизации в HTTP
  • работы с REST и JSON API
  • анализа HTTP-трафика с помощью DevTools и Wireshark
